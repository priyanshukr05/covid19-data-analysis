# covid19-data-analysis


This project analyzes global COVID-19 data (confirmed cases and deaths per day)
using the WHO Covid public dataset.

## Features

- Loads daily global COVID-19 data directly from WHO site
- Cleans and prepares data for analysis
- Plots cumulative cases, daily new cases, deaths per 100k
- Creates an area chart and a heatmap
- Includes a Streamlit app (`app.py`) for interactive exploration

## Data Source

WHO - COVID-19 dataset  
https://srhdpeuwpubsa.blob.core.windows.net/whdh/COVID/WHO-COVID-19-global-daily-data.csv
